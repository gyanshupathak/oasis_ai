# Background Music

**Paste your downloaded MP3 files here.** The scene planner picks a `music_mood` based on the post (e.g. inspiring, reflective). Name your file to match:

**Folder path:** `d:\Oasis\assets\music\`

**File names (use one of these):**
- `inspiring.mp3` – uplifting, hopeful (good for resilience/human spirit posts)
- `reflective.mp3` – thoughtful, calm
- `calm.mp3`
- `hopeful.mp3`
- `ambient.mp3` – fallback when no mood-specific file exists
- `motivational.mp3`
- `nostalgic.mp3`
- `urgent.mp3`

**Example:** For your delivery partner / resilience post, the planner will likely pick `inspiring`. Save your downloaded track as:

```
d:\Oasis\assets\music\inspiring.mp3
```

Or use `ambient.mp3` – it's tried as a fallback when the chosen mood file is missing.

**Volume:** Music is mixed at 8% so the voiceover stays primary.
